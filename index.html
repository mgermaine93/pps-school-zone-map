About This Project

This map visualizes Pittsburgh Public Schools (PPS) attendance zones, showing which schools each residential address is assigned to across elementary, middle, and high school levels.

Each dot represents a geocoded address in the PPS district. Colors indicate the assigned school based on the currently selected Color by dimension.

How to use

  • Color by — switch between Elementary/K8, Middle, and High School coloring
  • Filter by type — show only addresses zoned for a given school level
  • Filter by school — highlight all addresses assigned to a specific school
  • Search — type an address to zoom to it and see its school assignments
  • Legend — click any school to isolate its addresses

Data

Address and school zone data sourced from Pittsburgh Public Schools. Map tiles by OpenStreetMap contributors.
